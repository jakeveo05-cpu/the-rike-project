### 5. Handle Objections

| Objection | Response |
|-----------|----------|
| **"It's too expensive"** | "I understand budget matters. What would [specific result] be worth to your business? If we could [outcome] in 30 days, would that justify the investment?" |
| **"I need to think about it"** | "Totally fair. What specifically would you need to think through? Maybe I can help answer that now." |
| **"I need to talk to my partner"** | "Of course. Would it help if I joined a quick call with both of you? I can answer any questions they have." |

### 6. Close (2 min)

```

```

