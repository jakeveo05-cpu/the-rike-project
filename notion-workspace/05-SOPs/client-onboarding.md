## Overview

**Duration**: 7-14 days
**Owner**: Strategist (lead), All team (support)
**Trigger**: Contract signed & payment received

---

## üìã Onboarding Checklist

### Day 0: Contract Signed

- [ ] Confirm payment received
- [ ] Send welcome email
- [ ] Send onboarding form (Google Form / Typeform)
- [ ] Create client card in Notion
- [ ] Create client Slack channel (if applicable)
- [ ] Notify team in #clients channel
- [ ] Schedule kickoff call (within 48 hours)

### Day 1-2: Information Gathering

- [ ] Onboarding form completed by client
- [ ] Collect all access credentials:
  - [ ] Website admin
  - [ ] CRM access
  - [ ] Social media accounts
  - [ ] Google Analytics
  - [ ] Google Business Profile
  - [ ] Ad accounts
- [ ] Verify all credentials work
- [ ] Document in client card

### Day 2-3: Kickoff Call

**Before call**:
- [ ] Review onboarding form responses
- [ ] Prepare kickoff agenda
- [ ] Prepare initial questions

**During call (45-60 min)**:
- [ ] Introductions & team roles
- [ ] Review goals & KPIs
- [ ] Confirm scope of work
- [ ] Discuss communication preferences
- [ ] Set expectations for first 30 days
- [ ] Schedule first check-in (Week 2)

**After call**:
- [ ] Send meeting summary email
- [ ] Create tasks in system
- [ ] Assign team members

### Day 3-7: Initial Setup

**Architect**:
- [ ] Audit current systems
- [ ] Set up CRM/automation
- [ ] Configure integrations
- [ ] Set up tracking

**Creator**:
- [ ] Audit current content
- [ ] Create content strategy
- [ ] Plan first month content
- [ ] Set up content calendar

**Strategist**:
- [ ] Create reporting dashboard
- [ ] Set up KPI tracking
- [ ] Document baseline metrics

### Day 7-10: First Deliverables

- [ ] Initial audit report completed
- [ ] Strategy document shared
- [ ] First automation live
- [ ] First content pieces ready
- [ ] Client review & feedback

### Day 14: Check-in Call

- [ ] Review first 2 weeks
- [ ] Address any issues
- [ ] Confirm ongoing cadence
- [ ] Celebrate quick wins
- [ ] Set Month 1 goals
